3D Models
by Tyler Edwards 
Started November 2023

Description: I will upload various 3D model projects I work on, in .f3d, .3mf, .gcode, .obj, and .stl formats.

Each format:
- .f3d - Fusion Autodesk file
- .3mf - Prusa Slicer file
- .gcode - Sliced, Ready to Print file
- .obj / .stl - Object file
